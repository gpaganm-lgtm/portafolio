# Portfolio Profesional · Ginés Pagán

Este es un portafolio web estático, responsivo y moderno, diseñado específicamente para presentar un perfil técnico híbrido enfocado en **Soporte IT, Administración de Sistemas, Operaciones de Seguridad (SOC) y Mejora Continua**.

## 🎨 Diseño & Estética (Cyber-Ops Emerald Theme)
La interfaz utiliza una paleta premium oscura inspirada en software crítico de control y centros de monitoreo (SOC/NOC):
- **Fondo Principal:** Tono noche profundo (`#0a0e17`) que previene la fatiga visual.
- **Color de Acento:** Verde Esmeralda (`#10b981`) que simboliza un estado de sistema optimizado ("STATUS: OK") y hace alusión a la certificación **Lean Six Sigma Green Belt** del autor.
- **Tipografías:** Inter para textos legibles y JetBrains Mono para elementos de consola y etiquetas.

## 🛠️ Tecnologías Utilizadas
- HTML5 estructurado de forma semántica.
- CSS3 puro sin frameworks, aprovechando variables nativas, flexbox, grid y filtros de desenfoque de fondo (`backdrop-filter`).
- JavaScript Vanilla para interactividad básica (menú móvil, scroll fluido e IntersectionObserver para animaciones al hacer scroll).
- Iconografía técnica integrada directamente mediante un sprite SVG embebido.

## 📁 Archivos Incluidos
- `index.html`: Estructura principal y contenidos organizados.
- `styles.css`: Estilos visuales adaptativos y paleta de colores.
- `script.js`: Controladores de interacción y efectos dinámicos.

## 🚀 Despliegue en GitHub Pages
Para desplegar este portfolio de forma gratuita en tu cuenta de GitHub, sigue estos pasos:
1. Crea un repositorio público en GitHub llamado `tuusuario.github.io`.
2. Sube estos tres archivos (`index.html`, `styles.css`, `script.js`) directamente a la raíz de ese repositorio.
3. ¡Listo! En un par de minutos tu portafolio estará visible a nivel mundial en la URL: `https://tuusuario.github.io`.
